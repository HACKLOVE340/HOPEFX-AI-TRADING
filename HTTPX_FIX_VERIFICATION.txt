PROBLEM
-------

  "ERROR tests/integration/test_api.py - RuntimeError: 
   The starlette.testclient module requires the httpx package to be installed."

Exit Code: 2
Impact: CI tests could not run

ROOT CAUSE
----------
- tests/integration/test_api.py uses FastAPI TestClient
- TestClient depends on Starlette's test client
- Starlette's test client requires httpx package
- httpx was missing from requirements.txt

SOLUTION
